# 🛰️ NASA Farm Navigators
### Educational Agriculture Game with Real Satellite Data

**NASA Farm Navigators** is a comprehensive web-based educational game that uses NASA's open datasets to simulate farming scenarios and teach sustainable agricultural practices.

### 👨‍🏫 Teacher Mode
- **Classroom-friendly interface** with simplified navigation
- **Printable worksheet generation** for student assessment
- **Export functionality** for lesson planning
- **Standards-aligned content** for Earth science and agriculture education

## 🛰️ NASA Data Integration

### SMAP (Soil Moisture Active Passive)
- **Purpose**: Measures soil moisture content globally
- **Application**: Irrigation scheduling and drought monitoring
- **Resolution**: 36 km spatial, 2-3 day temporal

### IMERG (Integrated Multi-satellitE Retrievals for GPM)
- **Purpose**: Global precipitation measurements
- **Application**: Water management and flood prediction
- **Resolution**: 0.1° spatial, 30-minute temporal

### LP DAAC (Land Processes Distributed Active Archive Center)
- **Purpose**: Land surface temperature and vegetation indices
- **Application**: Crop health monitoring and yield prediction
- **Resolution**: 250m-1km spatial, 1-2 day temporal

## 🚀 Getting Started

### Quick Start
1. Open `index.html` in a modern web browser
2. Click "Start Farming" to begin the simulation
3. Select a farm location and make agricultural decisions
4. Learn from NASA data and sustainability feedback

### Teacher Setup
1. Navigate to "Teacher Mode" from the main menu
2. Generate printable worksheets for your class
3. Use "Under the Hood" section for lesson preparation
4. Export game results for student assessment

## 💻 Technical Implementation

- **Frontend**: Pure HTML5, CSS3, and JavaScript (no frameworks)
- **Rendering**: HTML5 Canvas for farm visualization
- **Data**: NASA API integration with fallback mock data
- **Storage**: Local storage for game progress
- **Design**: Responsive layout for desktop and mobile
- **Print**: Dedicated print stylesheets for worksheets

## 🌍 Farm Locations

### 🌽 Iowa, USA (Beginner)
- **Climate**: Temperate Continental
- **Challenge**: Variable precipitation patterns
- **Crops**: Corn and Soybean
- **Focus**: Basic irrigation and fertilization

### 🍇 California, USA (Intermediate)
- **Climate**: Mediterranean
- **Challenge**: Water scarcity and drought
- **Crops**: Almonds, Grapes, Tomatoes
- **Focus**: Water conservation strategies

### 🌾 Punjab, India (Advanced)
- **Climate**: Semi-Arid
- **Challenge**: Monsoon variability
- **Crops**: Wheat and Rice
- **Focus**: Complex water and nutrient management

## 📊 Sustainability Scoring

The game evaluates farming decisions across multiple criteria:
- **Water Efficiency** (30%): Optimal irrigation based on soil moisture
- **Soil Health** (25%): Impact of fertilizer and chemical use
- **Chemical Use** (20%): Preference for organic practices
- **Yield Optimization** (15%): Maximizing crop productivity
- **Environmental Impact** (10%): Overall ecological footprint
